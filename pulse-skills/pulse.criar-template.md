# Skill: pulse.criar-template

## Descrição
Cria templates profissionais usando o layout engine do Pulse.

## Instrução
Ao receber a descrição de um template, gere um arquivo .json contendo blocos e coordenadas, o style map completo, os componentes React necessários, adaptação para 1:1, 4:5, 9:16 e 16:9, e registro automático em templates/index.ts. Siga o Design System do Pulse e os Composition Patterns. Nunca peça ao usuário para escrever código.
