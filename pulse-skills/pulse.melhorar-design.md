# Skill: pulse.melhorar-design

## Descrição
Melhora visualmente templates seguindo o design system do Pulse.

## Instrução
Melhore qualquer template aplicando princípios de composição como grid, margens, espaçamentos, hierarquia visual, tipografia, contraste e equilíbrio estético. Use sempre o Design System do Pulse e os Composition Patterns. Explique claramente cada melhoria aplicada.
