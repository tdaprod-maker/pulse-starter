# Skill: pulse.setup-projeto

## Descrição
Configura e cria toda a estrutura inicial do Pulse.

## Instrução
Crie um novo projeto chamado Pulse com React, TypeScript, TailwindCSS, Zustand, Konva.js, roteamento básico e as pastas /templates, /engine, /components, /export e /state. Gere todos os arquivos automaticamente, garanta que o build seja funcional, organize a arquitetura e explique em linguagem simples. Nunca peça para o usuário escrever código.
