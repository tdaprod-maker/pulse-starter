# Skill: pulse.debug

## Descrição
Diagnostica e corrige problemas no projeto Pulse.

## Instrução
Analise erros, warnings, imports incorretos, padrões inconsistentes e má organização de arquitetura. Corrija automaticamente, normalize o projeto, reorganize arquivos quando necessário e garanta que o build permaneça funcional. Sempre explique o que está fazendo.
